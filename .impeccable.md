# Impeccable Design Context — Mini Agent

> 由 impeccable 技能维护。改界面前先读此文件，保持设计语言一致。

## 产品定位
- **是什么**：会自己调工具的中文智能体（聊天界面 + 文档理解 + 历史会话）。
- **目标用户**：中文知识工作者，需要"能办事"的 agent（算数/搜索/天气/笔记/长期记忆/多格式文档解析与跨文档比对）。
- **使用场景**：网页聊天；上传 PDF/Word/Excel/PPT/网页等做解析与比对；回看历史对话。
- **品牌调性**：未来感、赛博（保留赛博机器人 logo + "the future" 标语），但**界面本身要沉稳可信，不做噱头**。

## 设计方向：Editorial Calm（编辑感 · 沉稳）
- 内容优先、克制高级，参考 Claude / ChatGPT / Perplexity 的主流 agent 布局。
- **明确规避**「AI 味」：青+深紫渐变、霓虹光晕、满屏毛玻璃。
- 单一克制强调色（非霓虹），发丝级边框，柔和低透明阴影，指数缓出动效（无回弹）。

## 设计 Token（见 index.html :root）
- 强调色：浅色 `#5b54e6` / 深色 `#8b84f0`（沉稳靛紫，非渐变）。
- 中性底色：浅 `#f5f5f4`（暖纸感）/ 深 `#0c0c0e`（石墨，非纯黑）。
- 表面/边框：`--surface` / `--surface-2` / `--border`（发丝灰）。
- 字体：正文 system-ui + PingFang SC/Noto Sans SC；品牌字 Space Grotesk（display）；代码 JetBrains Mono 回退。
- 形状：圆角 `--r-sm/md/lg`，按钮 `--r-pill`。阴影 `--shadow-sm/md/lg` 低透明。
- 动效：`--ease` cubic-bezier(.16,1,.3,1)；`prefers-reduced-motion` 全量降级。

## 布局结构（sticky 全宽）
- 顶栏 `.topbar`（sticky，发丝底边，内容居中 920px）：品牌 + 历史/新对话/主题三态。
- 聊天区 `.chat`（居中 840px，独立滚动）。
- 底部 `.composer-wrap`（sticky bottom，渐隐背景）：附件面板 sheet + composer（📎 + 自适应 textarea + 发送箭头图标）+ 提示行。
- 历史抽屉 `.drawer.left`（左侧滑入）。

## 必保留的功能钩子（JS 依赖，勿删 class/id）
- 主题：`#themeSwitch`（light/dark/system，持久化 localStorage）。
- 聊天：`#chat #input #send`，`.composer.disabled` 由 JS 切换。
- 附件：`#attachPanel #attachBtn #uploadZone #fileInput #urlInput #docSearchInput #compareBar #cmpA #cmpB #compareRun #compareClear #docList`。
- 历史：`#histToggle #histDrawer #histBackdrop #histClose #histList`。
- 审核卡片：`.review #reviewCard #approveAll #rejectAll #reviewNote`，`.rbtns .ok/.no`，`.rfooter .all-ok/.all-no`。

## 可访问性基线
- 对比度目标 WCAG AA（4.5:1 正文）；焦点 `:focus-visible` 强调色描边。
- 图标按钮带 `aria-label`；交互目标 ≥ 40px。
- 移动端 <640px 单列、抽屉满宽、间距收紧。
